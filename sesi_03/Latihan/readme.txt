1. Bintang.java
    - Program ini dibuat untuk menampilkan Bintang
    - Untuk menampilkannya, program ini memanfaatkan For Loop sebanyak 6 x lalu membuat baris baru
    - Output dari program ini adalah menghasilkan bintang sebanyak 5 kolom dan 6 baris

2. MenampilkanNilai.java
    - Program ini dibuat untuk melakukan perulangan dari suatu nilai  dengan menggunakan For.
    - Perulangan for memiliki struktur (insialisasi, kondisi, increment/decrement ) 
    - Output akan menghasilkan nilai dari variabel j sebanyak 10 kali.

3. BilanganGanjil.java 
    - Program ini dibuat untuk menampilkan bilangan ganjil dari 1 - 20.
    - Perulangan for digunakan untuk mengulang angka sebanyak 20 kali.
    - Terdapat logika modulus untuk menentukan mana bilangan yang ganjil.
    - Output akan mengeluarkan bilangan ganjil antara 1 - 20.
