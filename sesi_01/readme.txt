8. List.java
    Penjelasan:
        - Pada program ini terdapat source code untuk membuat Array List.
        - Array List merupakan suatu array yang terindeks dan dapat di ubah ukurannya secara dinamis .
        - Array list dapat diimport dari package java.util dan Class ArrayList.

    Cara Menggunakan :
        - Jalankan program, maka sistem akan mebuat suatu objek list yang bernama buah. Buah tersebut akan diisi dengan
          nama-nama buah. 
        - Penambahan nama-nama buah/data, dapat dipanggil dengan memanggil method add()
        - Untuk melakukan pengambilan data dapat menggunakan method get(index)
        - ArrayList memungkinkan juga untuk melakukan perubahan data dengan method set(index, value), method tersebut 
          memerlukan 2 parameter yaitu index dan value yang akan menjadi pengganti data yang lama.
        - Untuk melakukan hapus data dapat memanfaatkan pemanggilan method remove()
        - Ukuran suatu list dapat di cek dengan menggunakan method size()
        - Isi di dalam list dapat dilakukan juga dilakukan perulangan

11. 10. HashMap.java
    Penjelasan:
        - Pada program ini terdapat source code untuk membaut HashMap.
        - HashMap dideklarasikan dengan pasangan key dan value baik dengan tipe yang sama maupun berbeda.

    Cara Menggunakan :
        - Jalankan program, maka sistem akan mebuat suatu objek HashMap yang bernama cars. Cars tersebut akan diisi dengan
          nama-nama merk mobil (key) dan jenis mobil (value). 
        - Penambahan nama-nama cars/data, dapat dipanggil dengan memanggil method put(key, value).
        - Pada HashMap dapat melakukan pengambilan data dengan memanggil method get(key).
        - Pada HashMap dapat melakukan penghapusan data dengan menggunakan method remove(key).
        - Perulangan dapat dilakukan dengan menggunakan method keySet() apabila ingin mengambil keynya saja, dan dapat menggunakan 
          values() apabila ingin mengambil values nya saja.
