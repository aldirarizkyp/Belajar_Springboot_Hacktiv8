3. OperatorPenugasan.java
    Penjelasan:
        - Pada program ini terdapat source code untuk memberi tugas suatu variabel untuk melakukan proses.
        - Pengisian nilai ( = )
        - Pengisian dan penambahan (+=)
        - Pengisian dan pengurangan (-=)
        - Pengisian dan perkalian (*=)
        - Pengisian dan pembagian (/=)
        - Pengisian dan sisa bagi/modulus (%=)
      
    Cara Menggunakan:
        - Jalankan program dan sistem menginstansiasi variabel a = 5 dan b = 10.
        - Pada blok penjumlahan terdapat b += a yang dapat juga ditulis b = b + a, sistem akan melakukan penjumlahan
          b = 10 + 5, sehingga didapatkan nilai dari variabel b = 15.
        - Pada blok pengurangan terdapat b -= a yang dapat juga ditulis b = b - a, sistem akan melakukan pengurangan
          b = 15 - 5, sehingga didapatkan nilai dari variabel b = 10.
        - Pada blok perkalian terdapat b *= a yang dapat juga ditulis b = b * a, sistem akan melakukan perkalian
          b = 10 * 5, sehingga didapatkan nilai dari variabel b = 50.
        - Pada blok pembagian terdapat b /= a yang dapat juga ditulis b = b / a, sistem akan melakukan pembagian
          b = 50 / 5, sehingga didapatkan nilai dari variabel b = 10.
        - - Pada blok sisa bagi terdapat b %= a yang dapat juga ditulis b = b % a, sistem akan melakukan sisa bagi
          b = 10 % 5, sehingga didapatkan nilai dari variabel b = 0.

    Penjelasan:

    Cara Menggunakan:
        a. Program Operator Pembanding 1

            - Di setiap pencetakan terdapat beberapa perbandingan diantaranya : 
              Sama dengan (==), tidak sama dengan (!=), lebih besar dari (>), kurang dari (<), lebih besar dari sama dengan (>=),

        b. Program Operator Pembanding 2
            - Jalankan program, maka sistem akan mencetak perbandingan antara 2 variabel nilaiA dan nilaiB, dimana nilai nilaiA = 12
              dan nilaiB = 4.
            - Lalu terdapat perintah untuk melakukan perbandingan dua variabel nilaiA dan nilaiB diantaranya yaitu
              nilaiA > nilaiB, nilaiA < nilaiB, nilaiA >= nilaiB, nilaiA <=nilaiB, nilaiA == nilaiB, dan nilaiA != nilaiB yang dimana
              setiap perintah tersebut disimpan di dalam variabel hasil.
            - Hasil akan menghasilkan nilai boolean True atau False sesuai dengan kedua nilai yang dibandingkan.
