1. MenentukanBilangan.java
    - Program ini bertujuan untuk mendapatkan rata-rata dari 3 angka, lalu tentukan angka yang terbesar dan terkecil
      diantara ketiganya.
    - Melakukan deklarasi variabel x , y , z, average, besar, dan kecil.
    - Melakukan instansiasi objek in untuk memanfaakan kelas Scanner agar dapat diinput sesuai keinginan.
    - Melakukan pembuatan input x , y , dan z dengan menggunakan scanner.
    - Pada bagian proses, terdapat perhitungan untuk menghitung rata-rata dari nilai x,y,z.
    - Lalu untuk menentukan bilangan terbesar dan terkecil, program ini memanfaatkan seleksi kondisi if else
    - if else dilakukan dengan mengisikan parameter yang membandingkan ketiga angka tersebut.
    - Setelah di proses, program akanmenampilkan nilai rata-rata, bilangan terbesar, dan bilangan terkecil.

2. ArrayMultidimensi.java
    - Program ini dibuat untuk menentukan kelas dan tempat duduk dengan memanfaatkan array multidimensi.
    - Melakukan deklarasi array multidimensi yang bernama arr[][].
    - Melakukan instansiasi objek in untuk memanfaakan kelas Scanner agar dapat diinput sesuai keinginan.
    - Pengisian dilakukan dengan memanfaatkan nested for. Perulangan for luar untuk indeks pertama dan Perulangan
      for dalam untuk indeks kedua. 
    - Setelah mengisikan semua isi array, dilakukan pencetakan dengan memanfaatkan nested for yang sama.

3. Diskon.java
    - Program ini dibuat untuk menentukan apakah suatu pembelian mendapatkan diskon atau tidak
    - Melakukan deklarasi variabel pembelian, diskon, harga_bayar.
    - Melakukan instansiasi objek in untuk memanfaakan kelas Scanner agar dapat diinput sesuai keinginan.
    - Membuat input untuk memasukkan total pembelian. 
    - Lalu membuat logika untuk menentukan apakah pembelian mendapatkan diskon atau tidak dengan memanfaatkan if else.
    - Pembelian akan mendapatkan diskon, apabila pembelian lebih besar sama dengan 1.000.000. Jika kurang dari itu, maka
      pembelian tidak mendapatkan diskon.
    - Apabila mendapatkan diskon, maka program akan masuk kedalam perhitungan harga setelah diskon
    - Sistem akan menampilkan output berupa Pembelian, Diskon, dan Harga Bayar(setelah diskon).

4. TahunKabisat.java
    - Program ini dibuat untuk menentukan suatu tahun merupakan tahun kabisat atau tidak.
    - Syarat : 
            a. Jika angka tahun itu habis dibagi 400, maka tahun itu sudah pasti tahun kabisat.
            b. Jika angka tahun itu tidak habis dibagi 400 tetapi habis dibagi 100, maka tahun itu sudah pasti bukan merupakan tahun kabisat.
            c. Jika angka tahun itu tidak habis dibagi 400, tidak habis dibagi 100 akan tetapi habis dibagi 4, maka tahun itu merupakan tahun kabisat.
            d. Jika angka tahun tidak habis dibagi 400, tidak habis dibagi 100, dan tidak habis dibagi 4, maka tahun tersebut bukan merupakan tahun kabisat.
    - Melakukan deklarasi variabel tahun
    - Melakukan instansiasi objek in untuk memanfaakan kelas Scanner agar dapat diinput sesuai keinginan.
    - Untuk menentukan tahun kabisat atau bukan, program ini memanfaatkan kondisi if else yang dimana di dalam parameter nya terdapat
      perhitungan aritmatika dengan menggunakan modulus(%).
    - Output akan keluar berdasarkan seleksi if else mana yang memenuhi kriteria dari tahun kabisat dan bukan tahun kabisat.

5. PointOfSale.java
    - Program ini dibuat untuk menentukan point of sale sederhana.
    - Melakukan deklarasi variabel beli, total, dan i.
    - Melakukan deklarasi array yang dapat mengisi indeks sebanyak 5. Array tersebut terdiri dari array kode, barang, quantity,
      harga, sub_total, dan diskon.
    - Melakukan instansiasi objek in untuk memanfaakan kelas Scanner agar dapat diinput sesuai keinginan.
    - Melakukan input beli dengan scanner untuk menentukan berapa jenis barang yang akan di beli
    - Melakukan perulangan for untuk mengisikan data array kode dan quantity. Perulangan akan dilakukan sebanyak < beli.
    - Selanjutnya masih di dalam for, terdapat switch case untuk menentukan case dari masing2 pilihan beli. Setiap case nya akan
      akan mengisikan data array pada array barang, harga, dan diskon.
    - Terdapat nilai default untuk memberikan nilai apabila inputan berada diluar case.
    - Setelah perulangan pengisian data array sudah dilakukan sebanyak jumlah barang yang akan dibeli, array tersebut akan dicetak.
    - Proses pencetakan array dilakukan dengan perulangan for yang didalamnya terdapat juga perhitungan untuk harga setelah diskon dan total bayar
      untuk semua barang.
    - Sistem akan mengeluarkan output seperti struk belanja sederhana.
